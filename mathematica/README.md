# Mathematica 笔记本

计算机符号系统 (CAS) 软件是数学学习路上的好帮手。

尤其在数学学习过程中，能做符号计算，数值计算，绘图的 3M 软件，能帮大家节约不少时间。参见 [cas_software.md](cas_software.md)
